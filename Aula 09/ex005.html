Selecionando

Podemos selecionar os elementos de uma DOM por:

  • Marca: getElementsByTagName()[]. Entre parenteses colocamos a tag que queremos pegar e entre colchetes o numero da ordem dele de 0 até o ultimo.
  • ID: getElementById(). Entre parenteses identificamos o id do HTML, como o nome é Element no plural só pegamos um elemento por isso não precisa de colchetes.
  • Nome: getElementsByNames()[]Entre parenteses usamos o nome do objeto e no colchetes o numero de ordem.
  • Classe: getElementsByClassName()[] Entre parenteses o nome da classe e Colchetes o numero de ordem
  • Seletor: querySelector()entre parenteses colocamos o codigo css Ex. documente.querySelector('div#msg'). Esse metodo é o mais atual e recomendado.

Comandos novos em JavaScript:

  • Alert - Cria alerta na tela
  • Confirm - Cria alerta e espera uma resposta negativa ou positiva
  • Prompt - Cria um alerta que aguarda uma entrada do usuario em forma de string.
  • var - cria variavel. Ex var nome = 5.
  • style.color - Muda a cor da letra. Ex. n.style.color = 'red'
  • style.background - Muda cor do fundo. Ex. n.style.background = 'red'
  • document.write - Escreve no documento html. Podemos escrever innerText(texto puro), innerHTML(texto com formatação HTML) e innerURL(URLs)
